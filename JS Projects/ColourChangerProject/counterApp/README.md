Description:

A simple JavaScript Counter App which updates the displayed value when user clicks on increment, decrement and reset button

Features:

1. Increment Count
2. Decrement Count
3. Reset Count
4. DOM manipulation using jS
5. Dynamic colour change based on value
6. basic css layout and centering

What I learnt:

I this project I learnt about the difference between APP state and UI.

The current App state is stored in count variable and DOM is updated using updateCount() function

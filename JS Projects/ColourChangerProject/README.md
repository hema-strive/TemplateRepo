Description

This is a simple Javascript project that changes colour when button is clicked

Features

1. Change box colour dynamically
2. display current selected colour
3. DOM Manipulation with JS
4. event handling with buttonclick

What I learned

1. DOM selection (getElementById)
2. Event Listening (addEventListener)
3. Updating styles dynamically
4. Basic layout and centering

Future Improvements

1. Add color picker
2. Add animation
3. Store last selected color
